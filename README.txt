DXF BLUEPRINT GENERATOR

What it does
- Open index.html in a browser
- Drop in a 2D DXF file
- Enter the real overall width and height
- The app scales the imported drawing to those dimensions
- It generates a clean blueprint preview with title block and dimensions
- Download the result as SVG or use Print / Save PDF from the browser

Best use
- Concept drawings
- Quick field blueprints
- Trailer doors
- Panels
- Flat parts
- Enclosures

Supported DXF entity types
- LINE
- CIRCLE
- ARC
- LWPOLYLINE
- POLYLINE / VERTEX

Important
- This is not a full CAD engine
- It works best with clean 2D DXF files
- Dimensioning is based on overall bounding box width and height
- For fabrication-grade output, verify dimensions before build

How to use
1. Extract the zip folder
2. Open index.html in Chrome or Edge
3. Drop in your DXF file
4. Enter real width and height
5. Edit title block fields
6. Download SVG or Print / Save PDF
